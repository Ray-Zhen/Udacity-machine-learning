编程练习3：定义衡量标准Â¶

如果不能对模型的训练和测试的表现进行量化地评估，我们就很难衡量模型的好坏。通常我们会定义一些衡量标准，这些标准可以通过对某些误差或者拟合程度的计算来得到。在这个项目中，你将通过运算决定系数 R2 来量化模型的表现。模型的决定系数是回归分析中十分常用的统计信息，经常被当作衡量模型预测能力好坏的标准。

R2的数值范围从0至1，表示目标变量的预测值和实际值之间的相关程度平方的百分比。一个模型的R2 值为0还不如直接用平均值来预测效果好；而一个R2 值为1的模型则可以对目标变量进行完美的预测。从0至1之间的数值，则表示该模型中目标变量中有百分之多少能够用特征来解释。模型也可能出现负值的R2，这种情况下模型所做预测有时会比直接计算目标变量的平均值差很多。

在下方代码的 performance_metric 函数中，你要实现：

  • 使用 sklearn.metrics 中的 r2_score 来计算 y_true 和 y_predict的R2值，作为对其表现的评判。
  • 将他们的表现评分储存到score变量中。

或

  • (可选) 不使用任何外部库，参考决定系数的定义进行计算，这也可以帮助你更好的理解决定系数在什么情况下等于0或等于1。

def performance_metric(y_true, y_predict):
    """计算并返回预测值相比于预测值的分数"""
    
    from sklearn.metrics import r2_score
    score = r2_score(y_true, y_predict)

    return score

def performance_metric2(y_true, y_predict):
    """计算并返回预测值相比于预测值的分数"""
    import numpy as np
    y_true = np.array(y_true)
    y_predict = np.array(y_predict)
    
    y_mean = sum(y_true)/len(y_true)
    SS_tot = sum((y_true - y_mean)**2)
    SS_reg = sum((y_predict - y_mean)**2)
    SS_res = sum((y_true - y_predict)**2)
    r2 = 1 - SS_res/SS_tot
    
    score = r2

    return score

问题 8 - 回答：Â¶

1.k折交叉验证法是将训练数据集划分为k个大小相似的互斥子集，每次使用k-1个子集的并集作为训练集，剩下的作为验证集，这样就可以进行k次训练和验证，最终返回的是这k个测试结果的均值。
2.GridSearchCV能够自动的遍历多种模型的参数组合，利用交叉验证法，选出效果最好的参数。
3.从cv_results中可以看到每个模型参数交叉验证过程中平均训练时间，平均计算得分时间，平均测试时间，平均训练时间，当前模型参数等。每次交叉验证的测试和训练分数以及测试和验证分数的标准差。
4.在网格搜索时如果不使用交叉验证法，会导致选出来的模型有可能产生过拟合，从而使泛化误差大，交叉验证通过将数据集分成训练数据和验证数据，通过若干轮的训练和测试后，选择在验证集上泛化误差最小的模型参数，这样可以很好的利用网格搜索选择最佳参数。在当数据随机性不够好而划分出现了极端的情况下，比如划分到训练集恰好都是positive类的，而划分到验证集的恰好都是negative类，这种情况下模型分数有可能会很低也有可能会很高。使用交叉验证法训练集和验证集都可以遍历一遍数据集，经过多次的训练和验证后取模型误差的平局值，可以减少对模型的评分误差。

def fit_model(X, y):
    """ 基于输入数据 [X,y]，利于网格搜索找到最优的决策树模型"""
    from sklearn.model_selection import KFold, GridSearchCV
    from sklearn.tree import  DecisionTreeRegressor
    from sklearn.metrics import make_scorer
    
    cross_validator = KFold(n_splits=10, random_state=42)
    
    regressor = DecisionTreeRegressor()

    params = {'max_depth':range(1,11)}

    scoring_fnc = make_scorer(performance_metric)

    grid = GridSearchCV(estimator=regressor, param_grid=params, scoring=scoring_fnc,  cv=cross_validator)

    # 基于输入数据 [X,y]，进行网格搜索
    grid = grid.fit(X, y)

#    print pd.DataFrame(grid.cv_results_) 
    # 返回网格搜索后的最优模型

在这个练习中，你将需要将所学到的内容整合，使用决策树算法训练一个模型。为了得出的是一个最优模型，你需要使用网格搜索法训练模型，以找到最佳的 'max_depth' 参数。你可以把'max_depth' 参数理解为决策树算法在做出预测前，允许其对数据提出问题的数量。决策树是监督学习算法中的一种。

  • 计算当前模型的交叉验证分数
  • 返回最优交叉验证分数对应的模型

def cross_val_score(estimator, X, y, scoring = performance_metric, cv=3):
    """ 返回每组交叉验证的模型分数的数组 """
    from sklearn.model_selection import KFold
    
    kf = KFold(n_splits=cv)
    scores = []   
    for train_index, test_index in kf.split(X):
        X_train, X_test = np.array(X)[train_index], np.array(X)[test_index]
        y_train, y_test = np.array(y)[train_index], np.array(y)[test_index]
        reg = estimator.fit(X_train, y_train)
        y_predict = reg.predict(X_test)
        scores.append( performance_metric(y_test, y_predict) )
#    scores = [0,0,0]
    return scores


def fit_model2(X, y):
    """ 基于输入数据 [X,y]，利于网格搜索找到最优的决策树模型"""
    from sklearn.tree import  DecisionTreeRegressor
    
    best_score = 0
    max_depth = range(1,11)
    
#   遍历max_depth里的参数值，利用不同的参数计算出交叉验证分数的均值 
    for i in max_depth:
        regressor = DecisionTreeRegressor(max_depth = i, random_state=0)
        scores = cross_val_score(estimator=regressor, X=X, y=y, scoring = performance_metric, cv=3)
        score_mean = np.array(scores).mean()  #计算交叉验证分数均值
#       找出最好的交叉验证分数和对应的模型
        if score_mean > best_score:
            best_score = score_mean
            best_estimator = regressor
            
    #最优交叉验证分数对应的最优模型
  #  best_estimator = None
    return best_estimator
